# Customer Retention Project

## Overview
This project analyzes customer retention patterns and provides actionable insights for improving customer lifetime value.

## Project Structure

- **data/** - Raw, cleaned, and database files
- **notebooks/** - Python analysis and feature engineering
- **sql/** - SQL queries for segmentation and analysis
- **powerbi/** - Dashboard visualizations
- **output/** - Reports and playbooks

## Getting Started

1. Install requirements: `pip install -r requirements.txt`
2. Add raw customer data to `data/raw/customers_raw.csv`
3. Run notebooks in order for data processing
4. Execute SQL queries for analysis
5. Generate dashboards and reports

## Key Analyses

- Customer Segmentation
- Geographic Patterns
- Promotion Impact
- Product Category Analysis
- Loyalty Program ROI
